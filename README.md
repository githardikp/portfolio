# Developer Portfolio

This is a personal developer portfolio built with **TanStack Start**, **React**, **TypeScript**, and **Tailwind CSS**.

## Running locally

```bash
# Install dependencies
npm install
# Start dev server
npm run dev
```

Open http://localhost:3000 in your browser.

## Build for production

```bash
npm run build
```

## DSA Learning Module (Work in Progress)

This new section of the portfolio demonstrates how core Data Structures & Algorithms solve real-world engineering problems.

### 🗺️ Overview
1. **/dsa** – Landing page listing real-life problems.
2. **/dsa/:problemId** – Detail page that breaks down the applicable DS&A technique, complexity analysis and step-by-step solution.

### 🚀 Features
- Curated catalogue of scenarios mapped to the best-fit algorithm or data structure.
- Clean, responsive UI built with the existing React + Tailwind stack.
- Friendly explanations aimed at bridging theory ↔️ practice.

### 🔧 Local Development
Nothing additional is required beyond the existing dev workflow:
```bash
npm install
npm run dev
```
Open http://localhost:3000/dsa to explore.

Contributions welcome! Add more problems in `src/data/dsaProblems.ts`. 🎉
